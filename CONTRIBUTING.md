# Contributing to homebrew-grazer

Homebrew tap for Grazer. Contributions welcome!

## How to Contribute

### Adding or Updating Formulas
1. Fork this repository
2. Create a branch for your change
3. Test the formula locally
4. Run `brew audit` and `brew style`
5. Open a Pull Request

### Formula Guidelines
- Follow Homebrew formula conventions
- Include proper checksums
- Test on Intel and Apple Silicon

### Reporting Issues
- Include macOS and Homebrew versions
- Show full error output
- Try `brew doctor` first

## Questions?

Open an issue on this repo.
